ASCII

American Standard Code for Information Interchange

What is ASCII?

The American Standard Code for Information Interchange (ASCII) is a character encoding standard for representing a set of 95 printable characters (letters of the alphabet, numbers and symbols) and 33 control characters (like DEL, ESC and SYN). It was first published in 1963 and is still in use as of today, being the foundation of modern Unicode. A nation-wide standard was needed in order to make communication and transfer of data possible. The Internet would be useful without ASCII, because if two people assigned different values (code points) to the same character, an "e" could become an "a" for another person. ASCII is essentially telling the computer what character should correspond to a binary value. Everyone comes across ASCII on a daily basis, almost all text you read on a phone or a screen uses ASCII encoding.

Why should i learn about ASCII?

I think that everyone should at least have a basic understanding of the systems used in their everyday life, especially if they depend on it (via job, school, entertainment etc.). View it as common knowledge, as it is like knowing where meat comes from: not crucial for survival, but still of relative importance.
